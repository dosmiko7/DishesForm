
# HexOcean Dishes Form

Form created for HexOcean.





## Run Locally

Make sure you have Node and npm installed (https://nodejs.org/en)

Clone the project

```bash
  git clone https://github.com/dosmiko7/DishesForm
```

Go to the project directory

```bash
  cd DishesForm
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
